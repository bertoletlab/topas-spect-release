# Security policy

OpenTOPAS-SPECT is research software (Monte Carlo simulation decks and OpenTOPAS extension classes). It
does not handle credentials, network services, or patient data, so its security surface is small.

If you believe you have found a security-relevant issue (for example, a code path that could execute
untrusted input unexpectedly), please report it privately to **alebertolet@gmail.com** rather than
opening a public issue. We will acknowledge your report and respond as promptly as we can.

For ordinary bugs and incorrect results, please use the normal [issue tracker](../../issues).
